The $Casho Token

The Casho Utility Token is the native ERC-20 Utility Token of the Casho Protocol and the decentralized applications built on it. The Casho Utility Token regulates, incentivizes, and rewards the various participants of the Protocol.

Token Utility

Incentives

As an incentive system, users shall be rewarded with Casho tokens in return for their participation in the ecosystem. It acts as a form of payment that removes friction in transactions and allows micro payments.

DAO Governance

The Casho token allows holders to actively participate and have a say in the Casho ecosystem. Token holders can create and vote on proposals as a DAO

Ecosystem Development

The Ecosystem Development Fund (EDF) supports applications that are using Casho protocol. The EDF is governed by the DAO
